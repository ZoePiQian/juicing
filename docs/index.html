新消费品牌忽略了时间

文中说，无数想起就觉得美好的品牌，做到今天的品牌影响力更是花了几十乃至几百年的时间。我略惊讶地读到，连Lululemon的品牌都注册成立于23 年前。 时间，积累了一次次的失败，检验创业者的热情、心性，检验产品的品质、创新，把红利抹平，留下认真做品牌的人，积累起信誉。 我喜欢这句话：「对产品/服务体验一致性的长期记忆就是信誉。」 这是无法速成的。立一个看上去很美的品牌不难，难的是产品的价值创造，是长期的研究、热爱，是企业经营、组织文化一点一滴的塑造本身。这些才是穿透时间的力量。 「存在越久的事物，大概率会存在更久。」原因我想就在于背后的力量源泉。对于产品和个人我想都是如此。愿你亦有收获。

也许你该找个人聊聊

这本书是今年读到的自我探索类型书籍中的五星✨ 强烈推荐给对自我探索有兴趣的朋友。 作者洛莉·戈特利布是美国的执业咨询师，从业前曾是从事《老友记》等经典剧集的编剧之一，因此这本书在专业、真实的基础上同样具备极强的可读性。 作者通过四个来访者的故事（傲慢的聪明制片人、身患绝症的新婚女教授、即将走入七十岁的老太太、酗酒的年轻女性），与咨询师自己的咨询故事，试图向读者展示：每个人都在面对相通底层的人生议题——亲密关系、控制、不确定性、死亡。而在解决人生困境的过程中，诉说与倾听、看见与被看见，有着比普通人想象中更强的力量。 由于书中故事脱胎于真实的咨询案例，每一位来访者的故事都有着复杂、立体的面向，每个角色都具有极强的生命力。角色的切面与我们共鸣，目睹他们在困境中挣扎、鼓起勇气直面问题，确实能够令读者走向自己的内心更深处。 还是得说，书籍无法解决我们的问题，正如心理咨询也无法解决我们的问题，问题是在觉知产生后，通过一次次的行动一点点被解决。 尽管事实是很令人沮丧的，但看见自己就是改变的开始。

为什么沃伦·巴菲特有那么多的时间去阅读和思考？

在过去一年里，对我影响最大的话语是瑞·达里欧所说的「排列优先顺序：尽管你几乎可以得到你想要的任何东西，但你不可能得到你想要的所有东西。」 在这篇文章里，沃伦·巴菲特展现了在排列优先顺序上的天才之处。他的「不惜一切代价也要避免」的清单深深印刻在我的心上。 我自己和据我观察下的很多人，都会专注于自己的前几个目标，然后间歇性地在其余目标上投入精力。而巴菲特却通过把次要目标放在「不惜一切代价也要避免」的清单上，从策略的高度确保自己真正专注于优先级最高的目标。 这是迈克尔·西蒙斯研究的巴菲特为了有更多的时间阅读和思考而采用的六大策略之一。希望这六大策略以及其背后的心智模型，对你也有启迪。

是醒来，不是退行

最近在研究集体创伤，然后看到了高老师的这篇文章。我很喜欢这个标题：「不是退行，而是醒来。」 在心理学的语境中，退行（regression)是一种相对比较简单的防御机制，是指人们在受到挫折或面临焦虑、应激等状态时，放弃已经学到的比较成熟的适应技巧或方式，而退行到使用早期生活阶段的某种行为方式，以满足自己的某些欲望。 如果你和你的咨询师彼此建立了相互信任的关系，并且你带着敏锐的觉察去捕捉你们之间动力的起承转合，你会发现——退行是一个特别容易出现的场景。你可能在ta面前会变得像个孩子，你可能会露出幼稚/无理取闹/调皮的一面，但你不知道为什么会这样。 结合能量和防御机制这两个视角来看，退行的发生其实是你的神经系统在和你玩「捉迷藏」的游戏。 你的神经系统就像是一个神秘而巨大的环绕式圆形图书馆。这个图书馆有很多层书，很多个书架，如果在里面时间待的久了可能会迷路。 想象一个场景，这个图书馆有很多层书，很多个书架，而每个创伤都有自己的坐标，它可能具体到某个书架的第几层，但有时候也会具体到某本书的第几页，甚至是第几行。 你和你的治疗师需要将你的神经系统当做穿越时光的工具，你们要一起回到那个可能只有三岁的孩子的内心——你那突如其来像孩子般那样哭泣和撒娇，是在邀请治疗师将你的神经系统当做穿越时光的工具，你们一起回到退行发生的那个时间点，你们要一起和那个三岁的孩子去创造出一个亲密的关系，把创伤下载下来，共同调节这样的频率（创伤）。 而最让人喜悦却无奈的一点是，这一切并不发生在过去，而是发生在当下。所以，不是退行，而是醒来。

肖知兴：只有自组织才是真正的组织能力

组织从进化形式来看可以分为两类，他组织和自组织。在做 Juicing Today 这款小而美的网页时，我一直在思考，怎样可以去促使一个自组织的形成。 如果一个系统靠外部指令而形成组织，就是他组织。如果不存在外部指令，系统按照相互默契的某种规则，各尽其责而又协调地、自动地形成有序结构，就是自组织。（自组织的概念是相对于他组织而言的） 从进化论的角度讲，“自组织”是指一个系统在“遗传”“变异”和“优胜劣汰”机制的作用下，其组织结构和运行模式不断地自我完善，从而不断提高其对于环境的适应能力的过程。 自组织能力背后，主要是一种看不见、摸不着的价值观层面的力量，这确实是一条非常反直觉的逻辑线。所以我一直好奇究竟如何从价值观的层面去构建这样的力量。 这篇文章从宏观的视角解答了我的很多疑惑，预计完整读完要四十分钟，推荐整块时间阅读。

组织动力学：个体张力与组织合力

京东红太阳

之前看潘乱老师提到这篇18年旧文，说行业里再难见到这种“老道消息类”文体。 拜读完确实写得好，文本上故事性、理论性、商业性俱佳，底层能力是对于信息素材的糅合应用，对于学术理论的融会贯通，对于表达方式的琢磨凝练。 写刘强东其人，写京东这家公司，作者是用了很多社会学理论来解释老刘为人处事的世界观和他在企业经营管理的商业实践，落在京东的内部组织变革，最后又反映到宏观商业格局中京东的态势变迁。 自下而上四层架构，但又交织融合在一起。一气呵成，看得舒畅过瘾。加上文本表达详略得当，开头的场景特写、中间的回忆片段闪回、结尾描写又致敬了《百年孤独》那个经典的开头，如评论所说“像看了一部电影”。
