# 策展人: Zac

淘宝宣战拼多多的前夜：吕晋杰、陈琪、徐易容和葛永昌的至暗时刻

一篇关于电商战争的长文，有一万多字。 拼多多这几年快速崛起，成为阿里巴巴的强力对手。而在拼多多崛起之前的故事，也是很有意思的。 楚楚街是主打9块9包邮的鼻祖，其实最早涉足综艺节目广告投放的产品，并利用其中的红利快速做大，但他们自身的产品并不能满足快速增长的用户，从而最终走向了失败。而在当时还是拼好货CEO的黄铮曾多次拜访楚楚街进行学习与交流。此后，我们在拼多多上看到9块9包邮，在电视台密集投放，解决货品与客户需求匹配的问题，显然，是学习了楚楚街，从而避免了重蹈覆辙的事情。 看完你能深刻理解到商业是残酷，成功经常建立在别人的失败经验之上。而阿里巴巴战略的失败(打击淘客)，腾讯的扶持，也让拼多多走得远了很多。
