# 佛学

解读《波若波罗蜜多心经》

我的左手常年佩戴一个银环，上面刻的就是心经。心绪不宁时，常常凝视银环上的字，然后对自己说：诸法空相。 《心经》的全称《般若波罗蜜多心经》，是大乘佛教中空和般若思想的经典，也是一部文字最短少，却诠理最深奥微妙的经典，弘一法师评价说：《心经》虽仅二百余字，但摄全部佛法。 银河子的这篇《解读心经》解的非常之究竟。之前我一直困惑到底什么是「观自在菩萨」，银河子把它解释为：「往上可以连接上自己的天线、可以感受到自己的佛性、可以体会得到本源的圆满；往下，可以在这个物质世界中游刃有余。」 时时刻刻去关照自己内心的觉，观照自己内心的圆满，在这个物质世界中生活时时刻刻感觉到很愉快、很通透、很顺畅，然后你的心灵可以时时刻刻地保持“觉”，什么觉？就是绝对地松（放松）、满（满足）、爱（喜爱）的觉。所以，可以时时刻刻感受到心灵上的圆满，又可以在这个物质世界中游刃有余，即为“观自在菩萨”。
