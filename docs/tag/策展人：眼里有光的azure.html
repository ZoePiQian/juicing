#策展人：眼里有光的azure

一篇文章说清我们和金钱的关系

所有人生命题几乎都需要带着一个框架去思考，财富命题也不例外。要聊清楚「钱」这个话题并不容易。Lawrence Yeo 提出了一个名为「金钱光谱」的思考框架，并用一万多字说明了自己的见解。 人生中任何一个特定时刻，都对应着这张金钱光谱上的某一点，而光谱本身大致分为三个阶段：生存 - 自由 - 权力。对于不同阶段的人来说，金钱具有不同的意义。 把金钱光谱和情绪一起看时，则更发人深省。如果你以匮乏和生存的视角来看待世界，那么钱只会放大这种匮乏感。但如果自由对你最重要，那么无论你拥有多少钱，都会感到充实。如果你最渴望的是权力和影响力，那么钱将推动你和它的关系朝着这个方向发展。 你认为你正处于这个光谱上的什么位置？你未来又想在哪个位置？是什么在阻碍你？是钱，还是你的心态？
